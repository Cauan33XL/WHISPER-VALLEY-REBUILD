# Whisper Valley Remake

**Whisper Valley Remake** é a recriação do universo original que inspirou *Sycamore Valley*. 

O projeto original de Whisper Valley foi um marco importante e, por um período, correu o risco de se tornar uma *lost media*. Como uma iniciativa de preservação e revitalização, este projeto foi criado para manter a obra original viva e acessível. Todo o código base foi reescrito do zero com tecnologias web modernas (**TypeScript, Vite e motor Phaser 3**), garantindo máxima estabilidade e fluidez, mas preservando 100% da essência, mecânicas e a proposta criativa do jogo original.

## 🚀 Como Executar o Jogo Localmente

Certifique-se de ter o [Node.js](https://nodejs.org/) instalado em sua máquina.

1. **Instale as dependências:**
   ```bash
   npm install
   ```

2. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```
   O jogo estará disponível no navegador, geralmente no endereço `http://localhost:3000`.

3. **Para compilar o jogo para Produção / Deploy:**
   ```bash
   npm run build
   ```
   O jogo minificado e otimizado será gerado na pasta `dist/`.
